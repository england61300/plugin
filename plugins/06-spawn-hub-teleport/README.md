# 6. Spawn, hub, warps, teleport requests

Scaffold plugin module for `spawn-hub-teleport`.

## Commands
- `/setspawn`: Set global spawn.
- `/spawn`: Teleport to spawn.
- `/hub`: Return to hub.
- `/warp`: Warp commands.
- `/rtp`: Random teleport.
- `/tpa`: Request teleport.
- `/tpahere`: Request teleport to you.
- `/tpaccept`: Accept teleport request.
- `/tpdeny`: Deny teleport request.
- `/pwarp`: Player warp commands.

## Build
```bash
mvn clean package
```
